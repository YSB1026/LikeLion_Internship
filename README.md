### Commit Convention

- **[feat]** : 새로운 기능 추가
- **[fix]** : 버그 수정
- **[hotfix]** : 배포 후 긴급 버그 수정
- **[refactor]** : 기능 변경 없는 코드 개선
- **[chore]** : 설정, 빌드, 폴더 정리
- **[docs]** : 문서 작성 및 수정
- **[test]** : 테스트 코드 추가/수정
